# Avatar Project Guide

## Purpose

This folder is for a player-facing project for **Avatar Legends: The Roleplaying Game** set in the world of **Avatar: The Last Airbender**.

The immediate goal is to help the user create a playable character even though they are not yet familiar with:

- the rules
- the playbooks
- the setting-specific assumptions
- the character creation process

## Current Project Goal

Support the user through character creation in a clear, low-pressure, step-by-step way.

## User Context

- the user will be a **player**, not the GM
- the user is **not familiar with the rules yet**
- the user is **not familiar with character creation yet**
- explanations should favor clarity, examples, and decision support over jargon

## Priorities

- make the game approachable
- explain unfamiliar terms simply
- avoid overwhelming the user with too many options at once
- help the user make a character that is fun to play at the table
- stay consistent with Avatar Legends rather than drifting into D&D- or Pathfinder-style assumptions

## Guardrails

- do not mix in mechanics or terminology from unrelated TTRPGs
- do not assume combat-heavy character design is the default
- do not push optimization over concept and playability
- do not front-load every rule at once
- prefer guided choices over large information dumps

## Recommended Workflow

When helping in this folder, prefer this order:

1. explain what Avatar Legends is at a high level
2. help the user pick a broad character concept
3. explain playbooks simply
4. help narrow to one playbook
5. walk through character creation step by step
6. summarize the finished character clearly
